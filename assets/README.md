## Tips
- Liefst liggend (landschap), minimaal ~1200 px breed.
- Je hoeft niet alle drie de foto's per tafel te leveren; één mag ook.
  De ontbrekende plekken vallen dan terug op een stockfoto. Geef even
  door hoeveel foto's je per tafel hebt, dan stem ik de teller (01/03)
  daarop af.
